Council tax prototypes

About this prototype

Prototypes allow us to test ideas. Unless specified they are not designs that are "approved" and they are not production level code. These are in the first instance experiements, stimulus to test.

These prototypes are created using the gov.uk prototype kit, which is used across UK government to test ideas. The kit carries the gov.uk design system, which carries agreed styles, compoents and patterns, developed over the last eight years across government.

This prototype sits on a platform called Heroku. The repo (the code) is stored on GitHub. Any updates to that code automatically upload to Heroku (it takes a couple of minutes).
