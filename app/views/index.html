v2

Progression of v1 journey, user isn't logged in

  • Start page split into two screens: 1. An overall Direct Debt page with timescale expectancies for the tax payer; 2. A dedicated start page for Set Up A Direct Debit Online
  • Providing a council tax account number on its own page. Also contains an example of error messaging.
  • Looks up property address and returns property address and asks if this is the property to pay council tax for.
  • Displays amount outstanding, ie. the amount the Direct Debit will be set up from. Also notes if there is an unsummonsed arrear amount
  • Ask user if they want to receive communications by email
  • Ask if council tax bill address is the same as home address, theory: Saves user entering an address, needed to set up Direct Debit
  • Updated, split out into sections, check information screen, which also displays data carried through
  • Updated confirmation screen with clearer what has happened and what happens next

v2 landing page

v3

Progression of v2 journey, after review with staff. User isn't logged in

  • Revision of start page, to include more detail on what you need before you start. Also included link to other methods of setting up a Direct Debit.
  • Now ask for property postcode to validate access to set up Direct Debit. (You need to enter SE10 1AA to continue.)
  • New screen stopping setting up a Direct Debit, asking user to check their reference/s and they are entering them correctly. If problem persists directs them to teh council tax team.
  • Slight redesign of amount due. Also have an option for account with arrears, directing the user to contact the council tax office.
  • Communication preferences only appears if email address provided
  • Payment date is now on its own screen to provide clearer guidance but also help work out options on the next screen
  • Rework of payment schedule screen: example 1, example 2, and example 3 with an odd first payment
  • Bank details has bank branch removed after referring to HMRC. Is it needed to manually create a Direct Debit though?
  • Updated view of payment schedule on check information screen, which also displays data carried through. Also now carries Direct Debit Guarantee content and some terms of service.
